印刷

XMind でマップを印刷することは簡単です。これらの手順にしたがうだけです。

  1. マップを選択します。
  2. いずれかの方法で、印刷プレビューダイアログを開きます。
    • メニューの「ファイル → 印刷」を選択します。
    • ショートカットキーの Ctrl + P を押します。
  3. 印刷設定を変更： コンテンツ、ページ設定、方向、余白、ヘッダー/フッター
  4. [次へ＞]をクリックし、印刷するプリンターを選択します。
  5. [OK]をクリックします。
